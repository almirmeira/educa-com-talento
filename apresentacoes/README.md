# Apresentações dos Cursos

Esta pasta contém as apresentações em PowerPoint (PPTX) de todos os cursos da plataforma Educa com Talento.

## Estrutura

```
apresentacoes/
├── metodologias-ensino/       # Curso: Metodologias de Ensino
│   ├── aula-01-metodologias-importancia.pptx
│   └── ... (demais aulas)
├── foco-produtividade/        # Curso: Foco e Produtividade nos Estudos
│   ├── aula-01-credibilidade-proposito.pptx
│   └── ... (demais aulas)
└── README.md
```

## Como Baixar

1. Navegue até a pasta do curso desejado
2. Clique no arquivo PPTX
3. Clique no botão "Download" ou "Raw"

## Como Editar

As apresentações foram criadas em formato PPTX padrão e podem ser editadas com:

- Microsoft PowerPoint (recomendado)
- Google Slides (importar arquivo)
- LibreOffice Impress
- Apple Keynote (importar arquivo)
