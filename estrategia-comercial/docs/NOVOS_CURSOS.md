# Novos Cursos Sugeridos - Expansão do Portfólio

## Visão Estratégica

Para alcançar a meta de R$ 200.000/mês, é fundamental expandir o portfólio de produtos. A estratégia é criar cursos que:

1. **Complementem** os cursos atuais
2. **Atendam** diferentes públicos-alvo
3. **Permitam** upsell e cross-sell
4. **Aumentem** o ticket médio e LTV

## Validação Antes do Lançamento

### Checklist de Validação

1. **Pesquisa de demanda**
   - [ ] Pesquisa com base de e-mails (Google Forms)
   - [ ] Análise de buscas (Google Trends, Ubersuggest)
   - [ ] Análise de concorrência (existe oferta similar?)

2. **MVP (Produto Mínimo Viável)**
   - [ ] Criar 2-3 aulas piloto
   - [ ] Oferecer para grupo beta (50 pessoas)
   - [ ] Coletar feedback e NPS

3. **Pré-lançamento**
   - [ ] Lista de espera (500+ interessados)
   - [ ] Webinar de apresentação
   - [ ] Early-bird com desconto

## Conclusão

A expansão do portfólio é **essencial** para:

1. **Aumentar ticket médio** através de combos e upsell
2. **Diversificar públicos** (concurseiros, corporativo, escolas)
3. **Criar recorrência** com programas anuais
4. **Fortalecer a marca** como autoridade em aprendizado

### Prioridades Imediatas

1. **Memorização e Aprendizado Acelerado** - maior demanda
2. **Inteligência Emocional** - complemento natural
3. **Combo Aprovação Total** - aumentar ticket médio
