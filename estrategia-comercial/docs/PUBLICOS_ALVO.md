# Mapeamento de Públicos-Alvo

## Visão Geral

O Educa com Talento atende **7 segmentos principais** de público, cada um com características, necessidades e canais de aquisição específicos.

        │                       │                       │

## 1. Estudantes de Ensino Médio e Pré-Vestibular

### Perfil

| Característica | Descrição |
|----------------|-----------|
| **Faixa etária** | 14-19 anos |
| **Tamanho do mercado** | ~8 milhões de estudantes |
| **Dor principal** | Não sabem estudar de forma eficiente |
| **Decisor de compra** | Pais (70%) / Próprio estudante (30%) |
| **Ticket médio aceitável** | R$ 97 - R$ 297 |

### Canais de Aquisição

| Canal | Estratégia | Custo Estimado |
|-------|------------|----------------|
| **Instagram/TikTok** | Conteúdo curto sobre técnicas | R$ 0,50-2,00/lead |
| **YouTube** | Vídeos longos + SEO | Orgânico |
| **Google Ads** | "Como estudar para ENEM" | R$ 1-3/clique |
| **Parcerias escolas** | Workshops presenciais | Comissão 20-30% |
| **Influenciadores estudantis** | Publipost/parceria | R$ 500-5.000/post |

### Como Contactar

**Direto ao estudante:**
- Perfis educacionais no Instagram (@estudaquepassa, @medway, etc.)
- Grupos de WhatsApp de turmas
- Comunidades Discord de estudantes
- TikTok com dicas rápidas

**Via pais:**
- Facebook (público mais velho)
- Grupos de pais no WhatsApp
- Escolas (reuniões de pais)
- E-mail marketing via escolas parceiras

### Produto Recomendado
- **Curso Foco e Produtividade** (atual) - R$ 197
- **Mini-curso gratuito** como isca (lead magnet)
- **Plano família** (acesso para até 3 pessoas) - R$ 347

## 6. Secretarias de Educação (Municipal/Estadual)

### Perfil

| Característica | Descrição |
|----------------|-----------|
| **Quantidade** | 5.570 municípios + 27 estados |
| **Decisor de compra** | Secretário + equipe técnica |
| **Dor principal** | IDEB baixo, formação de professores |
| **Ticket médio** | R$ 20.000 - R$ 500.000 |
| **Ciclo de venda** | 6-18 meses (licitação ou dispensa) |

### Canais de Aquisição

| Canal | Estratégia | Custo Estimado |
|-------|------------|----------------|
| **UNDIME** | União dos Dirigentes Municipais | Evento/parceria |
| **CONSED** | Conselho de Secretários Estaduais | Evento/parceria |
| **Licitações** | Monitorar portais de compras | R$ 200-500/mês software |
| **Indicações políticas** | Relacionamento | R$ 0 |
| **Projetos incentivados** | FUMCAD, CONDECA | Via captação |

### Como Contactar

1. **UNDIME regional:** Solicitar participação em eventos
2. **Secretarias diretamente:** E-mail para coordenação pedagógica
3. **Políticos:** Vereadores com bandeira educação
4. **Professores da rede:** Bottom-up (indicação)
5. **Conselhos Municipais de Educação:** Apresentações

### Modelo de Entrada

```
ESTRATÉGIA DE ENTRADA EM SECRETARIAS:

Passo 1: Piloto gratuito em 2-3 escolas municipais
         ↓
Passo 2: Apresentar resultados ao Secretário
         ↓
Passo 3: Propor formação para professores (pequena escala)
         ↓
Passo 4: Expandir via dispensa de licitação (até R$ 17.600)
         ↓
Passo 5: Projeto maior via licitação ou emenda parlamentar
```

## 7. ONGs, Fundações e Institutos

### Perfil

| Característica | Descrição |
|----------------|-----------|
| **Quantidade no Brasil** | ~820.000 OSCs |
| **Com foco em educação** | ~15% (~120.000) |
| **Decisor de compra** | Diretor/Coordenador |
| **Dor principal** | Metodologias eficientes, captação |
| **Ticket médio** | R$ 5.000 - R$ 100.000/projeto |

### Canais de Aquisição

| Canal | Estratégia | Custo Estimado |
|-------|------------|----------------|
| **Mapa das OSCs** | IPEA - base de dados | Gratuito |
| **GIFE** | Rede de fundações | Evento/parceria |
| **Editais** | Itaú Social, etc. | Submissão conjunta |
| **LinkedIn** | Prospecção terceiro setor | Orgânico |
| **Eventos** | Fórum Social, GIFE | Networking |

### Como Contactar

- **Mapa das OSCs (IPEA):** [mapaosc.ipea.gov.br](https://mapaosc.ipea.gov.br)
- **GIFE:** [gife.org.br](https://gife.org.br)
- **ABCR:** Associação Brasileira de Captadores de Recursos
- **LinkedIn:** Buscar "Diretor ONG educação"
- **Editais conjuntos:** Propor parceria para submissão

### Modelo de Parceria

```
PROPOSTA PARA ONGs/FUNDAÇÕES:

Opção 1: LICENCIAMENTO
• ONG usa nosso conteúdo com sua marca
• Valor: R$ 10.000/ano + R$ 20/aluno

Opção 2: PARCERIA EM EDITAIS
• Educa com Talento entra como parceiro técnico
• ONG como proponente
• Split: 60% ONG / 40% Educa com Talento

Opção 3: CAPACITAÇÃO DE EDUCADORES
• Formação dos educadores da ONG
• Valor: R$ 5.000 + R$ 300/educador
```
