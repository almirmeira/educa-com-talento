### 2.2 Checklist de Otimização On-Page

```
ANTES DE PUBLICAR - CHECKLIST SEO

URL:
☐ Curta e descritiva (max 60 caracteres)
☐ Contém palavra-chave principal
☐ Sem caracteres especiais ou acentos
☐ Usa hífens (não underscores)
   Exemplo: /como-estudar-melhor

TÍTULO (Title Tag):
☐ 50-60 caracteres
☐ Palavra-chave no início
☐ Inclui número (se aplicável)
☐ Inclui ano atual [2025]
☐ Desperta curiosidade

META DESCRIPTION:
☐ 150-155 caracteres
☐ Contém palavra-chave
☐ Inclui CTA implícito
☐ Diferenciador claro

CONTEÚDO:
☐ Mínimo 2.000 palavras (conteúdo pilar)
☐ Palavra-chave nos primeiros 100 palavras
☐ Densidade de keyword: 1-2%
☐ LSI keywords (termos relacionados)
☐ Parágrafos curtos (3-4 linhas)
☐ Subtítulos a cada 300 palavras
☐ Listas e bullets
☐ Citações de fontes confiáveis
☐ Links internos (3-5 por artigo)
☐ Links externos (2-3 por artigo)

IMAGENS:
☐ Alt text descritivo com keyword
☐ Nome do arquivo otimizado
☐ Compressão (max 100KB)
☐ Formato WebP quando possível
☐ Dimensões especificadas

SCHEMA MARKUP:
☐ Article schema
☐ FAQ schema (quando aplicável)
☐ HowTo schema (tutoriais)
☐ Review schema (comparativos)
```

## 3. Link Building Gratuito

### 3.1 Estratégias de Custo Zero

#### Guest Posting Educacional

```
PASSO A PASSO PARA GUEST POSTING

1. IDENTIFICAR ALVOS
   Buscar no Google:
   • "escreva para nós" + educação
   • "guest post" + aprendizado
   • "colabore conosco" + estudos
   • site:edu.br + "autor convidado"

2. QUALIFICAR SITES
   Critérios mínimos:
   ☐ DA (Domain Authority) > 20
   ☐ Tráfego orgânico > 1.000/mês
   ☐ Relevância para educação
   ☐ Publica conteúdo recente
   ☐ Permite links dofollow

3. ABORDAGEM
   Template de e-mail:

   Assunto: Ideia de artigo para [Nome do Site]

   Olá [Nome],

   Sou [seu nome], especialista em técnicas de estudo
   e fundador do Educa com Talento.

   Acompanho o [site] há algum tempo e adorei o
   artigo sobre [tema específico].

   Gostaria de contribuir com um artigo exclusivo
   sobre [tema relevante]. Algumas ideias:

   1. [Título 1] - [breve descrição]
   2. [Título 2] - [breve descrição]
   3. [Título 3] - [breve descrição]

   Posso enviar um rascunho para avaliação?

   Abraço,
   [Seu nome]

4. PRODUZIR CONTEÚDO
   • Artigo de alta qualidade (1.500+ palavras)
   • Único e original
   • 1-2 links contextuais para seu site
   • Seguir diretrizes do site

5. FOLLOW-UP
   • Esperar 5-7 dias
   • Enviar lembrete educado
   • Máximo 2 follow-ups
```

#### Broken Link Building

```
ESTRATÉGIA DE LINKS QUEBRADOS

1. Encontrar links quebrados em sites educacionais:
   • Usar extensão "Check My Links" (Chrome)
   • Ou "Broken Link Checker" (Firefox)

2. Processo:
   a) Visitar sites de educação (.edu.br, universidades, blogs)
   b) Rodar verificador de links
   c) Identificar links 404 relevantes
   d) Criar conteúdo substituto em seu site
   e) Contatar webmaster oferecendo seu link

3. Template de contato:

   Assunto: Link quebrado encontrado em [página]

   Olá [Nome],

   Estava pesquisando sobre [tema] e encontrei
   seu excelente artigo sobre [título].

   Notei que o link para [recurso quebrado] não
   está funcionando mais (erro 404).

   Coincidentemente, publiquei recentemente um
   guia completo sobre o mesmo tema:
   [seu link]

   Caso seja útil, fique à vontade para usar
   como referência alternativa.

   Abraço,
   [Seu nome]
```

## 4. SEO Técnico (Custo Zero)

### 4.1 Core Web Vitals

```
MÉTRICAS ESSENCIAIS

┌─────────────────────────────────────────────────────────────────────┐
│                      CORE WEB VITALS                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  LCP (Largest Contentful Paint)        Meta: < 2.5s               │
│  ├── Otimizar imagens (WebP, lazy load)                            │
│  ├── Minificar CSS/JS                                               │
│  └── Usar CDN                                                       │
│                                                                     │
│  FID (First Input Delay)               Meta: < 100ms              │
│  ├── Reduzir JavaScript de terceiros                               │
│  ├── Otimizar execução de scripts                                  │
│  └── Usar web workers para tarefas pesadas                         │
│                                                                     │
│  CLS (Cumulative Layout Shift)         Meta: < 0.1                │
│  ├── Definir dimensões de imagens/vídeos                           │
│  ├── Reservar espaço para ads                                      │
│  └── Evitar inserção dinâmica de conteúdo                          │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

Ferramentas de teste (gratuitas):
• PageSpeed Insights (Google)
• GTmetrix
• WebPageTest
• Lighthouse (Chrome DevTools)
```

### 4.2 Checklist Técnico

```
AUDITORIA SEO TÉCNICA

INDEXAÇÃO:
☐ Sitemap.xml criado e atualizado
☐ Robots.txt configurado corretamente
☐ Páginas importantes indexadas
☐ Páginas desnecessárias em noindex
☐ Canonical tags em páginas duplicadas

ESTRUTURA:
☐ HTTPS ativo em todas as páginas
☐ Redirecionamentos 301 corretos
☐ Sem erros 404 (links internos)
☐ Breadcrumbs implementados
☐ Estrutura de URL limpa

MOBILE:
☐ Design responsivo
☐ Viewport configurado
☐ Botões touch-friendly (48x48px mín)
☐ Texto legível sem zoom
☐ Mobile-first indexing pronto

VELOCIDADE:
☐ Imagens comprimidas (< 100KB)
☐ CSS/JS minificados
☐ Browser caching ativo
☐ GZIP habilitado
☐ CDN configurado (Cloudflare gratuito)

SCHEMA MARKUP:
☐ Organization schema
☐ Course schema (para cursos)
☐ FAQ schema (páginas de dúvidas)
☐ Article schema (blog posts)
☐ Review schema (testimonials)
```

### 4.3 Implementação de Schema Markup

```json
// EXEMPLO: Course Schema para página de curso
{
  "@context": "https://schema.org",
  "@type": "Course",
  "name": "Foco e Produtividade nos Estudos",
  "description": "Curso completo de técnicas de estudo e produtividade baseadas em neurociência",
  "provider": {
    "@type": "Organization",
    "name": "Educa com Talento",
    "sameAs": "https://educacomtalento.com.br"
  },
  "hasCourseInstance": {
    "@type": "CourseInstance",
    "courseMode": "online",
    "courseWorkload": "PT30H",
    "inLanguage": "pt-BR"
  },
  "offers": {
    "@type": "Offer",
    "price": "297.00",
    "priceCurrency": "BRL",
    "availability": "https://schema.org/InStock"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "150"
  }
}
```
