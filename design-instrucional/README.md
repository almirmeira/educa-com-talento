## Sobre esta Vertical de Negócios

A **Educa com Talento** (operada juridicamente pela **Meira e Sousa Ltda.**) expande sua atuação para o mercado B2B de **Design Instrucional** — um dos segmentos de maior crescimento em educação corporativa e institucional no Brasil e no mundo.

Atuamos em toda a cadeia de valor da educação:

```
DIAGNÓSTICO → ESTRATÉGIA → PROJETO PEDAGÓGICO → PRODUÇÃO → IMPLEMENTAÇÃO → OPERAÇÃO → AVALIAÇÃO
```

## O Que Fazemos

| Área de Serviço | Descrição |
|---|---|
| **Universidades Corporativas** | Estruturação do zero ao funcionamento pleno — diagnóstico, governança, tecnologia, conteúdo e métricas |
| **Programas de Pós-Graduação EAD** | PPC completo, matriz curricular, DI de disciplinas, configuração de AVA e tutoria |
| **Plataformas AVA/LMS** | Seleção, implantação, customização, integração e operação gerenciada |
| **Design Instrucional de Cursos** | ADDIE, SAM, LXD — roteiro, storyboard, desenvolvimento, gamificação |
| **Projetos Pedagógicos** | PPC, PDI, PPP, diretrizes curriculares e adequação MEC/INEP |
| **Assessoria e Advisoring** | Mentoria estratégica para CLOs, coordenadores e IES |
| **Consultoria Educacional** | Diagnóstico institucional, adequação regulatória, transformação digital da aprendizagem |

## Por Que o Mercado de Design Instrucional?

| Dado | Valor | Fonte |
|---|---|---|
| Mercado global de LMS (2032) | USD 100,7 bilhões | MarketsandMarkets, 2026 |
| CAGR do mercado global de LMS | 18,4% ao ano | MarketsandMarkets, 2026 |
| Mercado global Smart Learning 2030 | USD 178,6 bilhões | MarketsandMarkets, 2026 |
| Investimento T&D por colaborador/ano (BR) | R$ 1.800–3.500 | ATD Research + ABTD |
| Empresas com cultura de aprendizagem têm | 218% mais receita por funcionário | ATD Research |
| Brasil no ranking EAD global | 3° maior mercado | ABED Censo EAD.BR 2025 |
| Crescimento EAD corporativo no Brasil | +25% ao ano | ABED, 2025 |

> **A oportunidade:** Empresas com 200 a 5.000 colaboradores precisam estruturar UCs e digitalizar treinamentos, mas não têm expertise interna. Este é o segmento mais acessível e de maior volume de clientes em potencial.

## Padrões e Normas Técnicas Suportados

| Standard | Descrição |
|---|---|
| **SCORM 1.2 / 2004** | Padrão universal de comunicação conteúdo ↔ LMS |
| **xAPI (Tin Can / IEEE 9274.1.1-2023)** | Rastreamento de qualquer experiência de aprendizagem |
| **cmi5** | Perfil moderno de xAPI para integração com LMS |
| **LTI (Learning Tools Interoperability)** | Integração de ferramentas externas ao AVA |
| **Quality Matters (QM)** | Rubrica internacional de qualidade para cursos online |
| **WCAG 2.2 AA** | Acessibilidade para conteúdo digital |
| **ISO 21001:2018** | Sistema de gestão para organizações educacionais |
