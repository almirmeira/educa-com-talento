## Visão Geral

Este diretório contém **10 cases completos** de Design Instrucional que demonstram a amplitude e profundidade dos serviços da Educa com Talento. Os cases são documentos de referência que servem como:

1. **Portfólio comercial** — demonstração de capacidade técnica para prospects
2. **Metodologia de trabalho** — como executamos cada tipo de projeto
3. **Referência interna** — guia para nossa equipe na condução de projetos similares

## Como Usar Estes Cases em Vendas

### Apresentação para Prospects
1. Identifique o tipo de projeto do prospect (UC, pós-grad, curso único, etc.)
2. Selecione o case mais próximo do contexto do cliente
3. Use os dados de resultado (ROI, métricas) como argumento de valor
4. Adapte a proposta comercial com o mesmo nível de detalhe

### Pontos de Destaque por Case

```
UNIVERSIDADE CORPORATIVA → Use dados de: 218% mais receita por funcionário
                            ROI calculado, Kirkpatrick L4, benchmark ATD

PÓS-GRADUAÇÃO EAD → Use dados de: conformidade MEC, taxa de completude 87%
                     NPS do curso, benchmark IES premium

CASES PONTUAIS → Use dados específicos de ROI de cada case
```
