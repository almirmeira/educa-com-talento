## Sumário Executivo

O **Instituto Dinâmica Digital (IDD)** é uma IES privada com credenciamento MEC para EAD, especializada em educação na área de Tecnologia e Transformação Digital. Com 3.200 alunos em cursos de graduação EAD, o instituto identificou uma oportunidade estratégica: criar uma linha de **pós-graduação lato sensu 100% online** voltada para profissionais que precisam atualizar-se em tecnologia e gestão.

O desafio era estruturar um programa de pós-graduação de alta qualidade acadêmica, diferenciando-se das commodities do mercado (aulas gravadas + prova), com metodologias ativas, identidade própria e posicionamento premium, aprovado pelo MEC e pronto para atrair e reter alunos em um mercado extremamente competitivo.

A Educa com Talento foi contratada para o **design instrucional completo do programa**: PPC, matriz curricular, design de todas as disciplinas, configuração do AVA e sistema de tutoria e avaliação.

## Dados do Projeto

| Item | Dado |
|---|---|
| **Cliente** | Instituto Dinâmica Digital (IDD) — empresa fictícia |
| **Tipo de IES** | Privada com credenciamento MEC para EAD |
| **Curso** | MBA em Transformação Digital e Inovação Empresarial |
| **Modalidade** | 100% EAD Online |
| **Carga horária** | 360 horas |
| **Duração** | 18 meses |
| **Vagas anuais** | 800 alunos |
| **Público-alvo** | Gestores, líderes e profissionais de TI (25–45 anos) |
| **Posicionamento** | Premium — metodologia ativa + network + projeto real |
| **Investimento DI** | R$ 120.000 (projeto + produção) + R$ 8.000/mês (operação) |
| **Prazo do projeto** | 5 meses (PPC + DI + AVA configurado + 1ª turma lançada) |
| **Plataforma** | Moodle 4.1 (já licenciado pelo IDD) |
| **Padrões** | SCORM 1.2 + H5P interativo + videoaulas no Panopto |

## Contexto: O Mercado de Pós-Graduação EAD

```
PANORAMA DO MERCADO (Brasil, 2026):
──────────────────────────────────────────────────────────────────
• Pós-graduação EAD: 3,5 milhões de matrículas ativas (INEP 2025)
• Crescimento: +22% ao ano
• Ticket médio MBA/Especialização online: R$ 300–600/mês
• Duração típica: 12–24 meses
• Principais players: FGV Online (premium), Uninter, Anhanguera,
  Estácio, Faculdade Única, FIA Online, Insper EAD

DIFERENÇA ENTRE COMMODITY E PREMIUM:
──────────────────────────────────────────────────────────────────
Commodity: videoaulas + provas + pouca interação = NPS 20–35
Premium: metodologia ativa + tutoria humanizada + projeto real +
         rede de contatos = NPS 55–75

ONDE O IDD ESTAVA:
• Nenhum programa de pós-graduação ainda
• Credenciamento para criar cursos sem aprovação prévia do MEC
• Oportunidade: posicionamento premium no nicho de TI
```
